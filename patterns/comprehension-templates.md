# Reusable Comprehension Templates

**Template 1: Full Mental Model (Healthcare Example)**

```text
Build [feature] for [domain].
Must respect [compliance: GDPR, HIPAA…].
Priorities: [privacy > speed | scale | safety].
Handle [scale story: 10k RPS bursts].
Failure modes: [list].
Success looks like: [metrics + user impact].
Output: code + tests + docs + telemetry.
```

**Template 2: Threat-Model-First**
```text
[Task].
Threat model: [list].
Priorities & constraints: [list].
Generate: full implementation + security notes.
```

**Template 3: Scale-Story-Driven**
```text
Implement [system] that must handle [scale story].
Failure modes to consider: [list].
Success looks like: [metrics + user impact].
Prioritize: [list].
```

Copy → paste → adapt → iterate in the notebook.

The more you customize these, the more value you extract.
